This item pushes the effected target directly away from source by some value. In cases of ambiguity, the target chooses.

If a creature is Pushed into an [[Object]] or [[Rules/Characters/index|Character]], both take 2 [[Piercing]] damage per remaining Push which ignores [[Deflection]]. If the Object or Character is destroyed or dies, any remaining Push continues to Push the target through.

Multiple push tags stack.
