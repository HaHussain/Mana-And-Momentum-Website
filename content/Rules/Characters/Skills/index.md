There are 12 skills in the game that represent different actions commonly taken by characters that may have a chance of failure.
A [[Rules/Characters/index|Character]] can have up to 3 proficiency points in each Skill.
Each proficiency point allows the [[Rules/Characters/index|Character]] to add 1 x [[Level Bonus]], 1.5 x [[Level Bonus]], 2 x [[Level Bonus]] respectively.

A character can only add a 3rd proficiency point to a skill upon reaching Level 15.
