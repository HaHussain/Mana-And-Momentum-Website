Infiltration is the skill of getting into places you're not supposed to with practised skill and knowledge.


# [[Strength]]

The [[Strength]] Infiltrate skill represents a character carefully breaking an object with their might to allow them access, kicking a locked door at the lock so the lock shatters, targeting the hinges on a treasure chest. It is a careful application of might.


# [[Dexterity]]

The [[Dexterity]] Infiltrate skill represents getting into a place through finesse and agility, whether that be hiding in shadows to apply stealth or using lockpicks to open doors and disable traps.


# [[Intelligence]]

The [[Intelligence]] Infiltrate skill represents breaking through the security of technical and magical systems, destroying glyphs to access the core of system to change it. Hacking.
